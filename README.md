# mbot

A Rust project with Tokio async runtime.

## Prerequisites

- Rust toolchain (install via [rustup](https://rustup.rs/))
- Cargo (comes with Rust)

## Installation

```bash
cargo build
```

## Running

```bash
cargo run
```

## Dependencies

- `tokio` - Async runtime with full features

## Development

```bash
# Build
cargo build

# Run
cargo run

# Run tests
cargo test

# Check code
cargo check
```
